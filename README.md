# My Android App

An Android application with Firebase integration, location services, and ad monetization.

## Features
- Firebase Authentication
- Firebase Analytics
- Location Services
- AdMob Integration
- Google Play Services

## Setup Instructions
1. Clone this repository
2. Open in Android Studio or VS Code with Flutter plugin
3. Run `flutter pub get` to install dependencies
4. Connect a device or start an emulator
5. Run the app with `flutter run`
